# 步骤 6：编码开发

## 前置条件

- 技术方案已确认（technical-solution.md 已存在，执行表"技术评审"= "完成"）
- 对应开发字段（前端开发/后端开发）≠ "完成"

## 做什么

1. **提议状态变更**（→ tripo-tables）：对应开发字段 → "进行中"

2. **加载 `tripo-dev` skill**（从"编码前确认"开始执行）

3. **实时更新 STATUS.md**（→ tripo-task-dirs）

4. **提议状态变更**（→ tripo-tables）：对应开发字段 → "完成"

## 如何定义完成

- [ ] tripo-dev 完成 Checklist 全部通过
- [ ] STATUS.md 已更新

# 步骤 9：发布上线

## 前置条件

- 测试验收通过
- 执行表状态 = "测试中"，需求池状态 = "验收/提测中"

## 做什么

1. **等待用户发布上线**

2. **确认上线完成**：
   - 用户告知已上线
   - 或通过其他方式确认

3. **提议状态变更**：
   - 执行表状态 → "完成"
   - 需求池状态 → "已完成"
   - 实际交付时间 → 当前日期

4. **更新 STATUS.md**：
   - 状态标记为 ✅ 已完成
   - 添加完成记录

## 如何定义完成

- [ ] 用户确认已上线
- [ ] 已提议状态为"完成"
- [ ] 实际交付时间已更新
- [ ] STATUS.md 已更新为完成状态

## 最终状态

| 表格 | 状态字段 | 值 |
|------|---------|-----|
| 需求池 | 需求状态 | 已完成 |
| 执行表 | 状态 | 完成 |

# 字段映射参考

## 产品需求池字段

| 字段名 | 类型 | 说明 |
|--------|------|------|
| 需求Owner | user[] | 需求负责人 |
| 研发Owner | user[] | 研发负责人 |
| 创建人 | user[] | 创建人 |
| Member | user[] | 团队成员 |
| 需求描述 | text | 需求简述 |
| 需求状态 | select | 未启动/开发交付中/已完成/验收提测中/暂停 |
| 绝对优先级 | select | L1-L5/SSS紧急插入 |
| 需求池 | select | Studio-营收/用户增长/用户满意度/基建类等 |

## 执行中需求字段

| 研发Owner | user[] | 研发负责人 |

## 状态值对照

### 产品需求池
未启动 / 定容确认 / 开发/交付中 / 验收/提测中 / 已完成 / 暂停

### 执行中需求
评审中 / 研发中 / 测试中 / 完成 / 风险 / 暂停

### 技术需求
未启动 / 开发/交付中 / 已完成
